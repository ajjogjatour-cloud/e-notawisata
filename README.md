# e-Nota Wisata

Aplikasi nota wisata berbasis HTML + PWA (Progressive Web App).

Dibuat untuk kebutuhan:
- Tour & travel
- Rental mobil
- Transport wisata
- Nota pembayaran customer
- Share WhatsApp
- Export JPEG

## Fitur

✅ Buat nota wisata langsung dari HP  
✅ Simpan pengaturan perusahaan  
✅ Upload logo perusahaan  
✅ Export nota menjadi JPEG  
✅ Kirim nota ke WhatsApp  
✅ Responsive mobile  
✅ Bisa dijadikan aplikasi Android (PWA)  
✅ Gratis hosting via GitHub Pages
